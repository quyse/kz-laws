# Қазақстан Республикасының Азаматтық кодексіне (Жалпы бөлім) өзгеріс пен толықтыру енгізу туралы

Қазақстан Республикасының Жоғарғы Кеңесі 1994 жылғы 27 желтоқсанда қабылдаған Қазақстан Республикасының Азаматтық кодексіне (Жалпы бөлім) (Қазақстан Республикасы Жоғарғы Кеңесінің Жаршысы, 1994 ж., N 23-24, (қосымша); 1995 ж., N 15-16, 109-құжат; N 20, 121-құжат; Қазақстан Республикасы Парламентінің Жаршысы, 1996 ж., № 2, 187-құжат; № 14, 274-құжат; № 19, 370-құжат; 1997 ж., № 1-2, 8-құжат; № 5, 55-құжат; № 12, 183, 184-құжаттар; № 13-14, 195, 205-құжаттар; 1998 ж., № 2-3, 23-құжат; № 5-6, 50-құжат; № 11-12, 178-құжат; № 17-18, 224, 225-құжаттар; № 23, 429-құжат; 1999 ж., № 20, 727, 731-құжаттар; № 23, 916-құжат; 2000 ж., № 18, 336-құжат; № 22, 408-құжат; 2001 ж., № 1, 7-құжат; № 8, 52-құжат; № 17-18, 240-құжат; № 24, 338-құжат; 2002 ж., № 2, 17-құжат; № 10, 102-құжат; 2003 ж., № 1-2, 3-құжат) мынадай өзгеріс пен толықтыру енгізілсін:

44-баптың 1-тармағы төртінші абзацының екінші сөйлемі алып тасталсын;

105-бап мынадай мазмұндағы 3-1-тармақпен толықтырылсын:

"3-1. Мемлекеттік мекеменің тиісті қаржы жылына арналып бекітілген қаржыландыру жоспарынан асып кететін шарттық міндеттемелер қабылдауына жол берілмейді.".
