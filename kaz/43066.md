# "Қазақстан Республикасында зейнетақымен қамсыздандыру туралы" Қазақстан Республикасының Заңына өзгерістер енгізу туралы

1-бап . "Қазақстан Республикасында зейнетақымен қамсыздандыру туралы" 1997 жылғы 20 маусымдағы Қазақстан Республикасының Заңына (Қазақстан Республикасы Парламентінің Жаршысы, 1997 ж., N 12, 186-құжат; 1998 ж. N 24, 437-құжат; 1999 ж., N 8, 237-құжат; N 23, 925-құжат; 2001 ж., N 17-18, 245-құжат; N 20, 257-құжат; 2002 ж., N 1, 1-құжат; N 23-24, 198-құжат; 2003 ж., N 1-2, 9-құжат; N 11, 56-құжат; N 15, 139-құжат; N 21-22, 160-құжат; 2004 ж., N 11-12, 66-құжат; N 23, 140, 142-құжаттар; 2005 ж., N 7-8, 19-құжат; N 11, 39-құжат; N 14, 55, 58-құжаттар; N 23, 104-құжат; 2006 ж., N 3, 22-құжат; N 8, 45-құжат; N 12, 69-құжат; N 23, 141-құжат; 2007 ж., N 2, 18-құжат; N 3, 20-құжат; N 4, 28, 30-құжаттар; N 9, 67-құжат; N 10, 69-құжат; N 24, 178-құжат; 2008 жылғы 24 қазанда "Егемен Қазақстан" және "Казахстанская правда" газеттерінде жарияланған "Қазақстан Республикасының кейбір заңнамалық актілеріне қаржы жүйесінің тұрақтылығы мәселелері бойынша өзгерістер мен толықтырулар енгізу туралы" 2008 жылғы 23 қазандағы Қазақстан Республикасының Заңы ; 2008 жылғы 25 қарашада "Егемен Қазақстан" және "Казахстанская правда" газеттерінде жарияланған "Қазақстан Республикасының кейбір заңнамалық актілеріне ұжымдық инвестициялау және жинақтаушы зейнетақы қорлары қызметінің мәселелері бойынша өзгерістер мен толықтырулар енгізу туралы" 2008 жылғы 20 қарашадағы Қазақстан Республикасының Заңы ) мынадай өзгерістер енгізілсін:

1) 13-баптың 4-тармағындағы "25 еселенген" деген сөздер "жиырма сегіз еселенген" деген сөздермен ауыстырылсын;

2) 14-баптың 3-тармағындағы "жиырма бес еселенген" деген сөздер "жиырма сегіз еселенген" деген сөздермен ауыстырылсын.

2-бап . Осы Заң 2009 жылғы 1 қаңтардан бастап қолданысқа енгізіледі.

Қазақстан Республикасының Президенті

Н. Назарбаев
