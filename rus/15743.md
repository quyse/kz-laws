# О внесении изменения в Указ Президента Республики Казахстан, имеющий силу закона, "О государственном предприятии"

Внести в Указ Президента Республики Казахстан, имеющий силу закона, от 19 июня 1995 г. N 2335 "О государственном предприятии" (Ведомости Верховного Совета Республики Казахстан, 1995 г., N 9-10, ст. 66; N 24, ст. 164; Ведомости Парламента Республики Казахстан, 1997 г., N 12, ст. 183; N 13-14, ст. 205; 1998 г., N 23, ст. 429; 1999 г., N 22, ст. 789; N 23, ст. 916; 2001 г., N 10, ст. 126; 2002 г., N 10, ст. 102) следующее изменение:

в подпункте 12) статьи 18 слово "кадастра" заменить словами "и земельного кадастров".

Президент Республики Казахстан
